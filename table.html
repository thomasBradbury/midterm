Terms Definitions
cd Change Directory, a way to navigate through the various folders the computer contains when using the command line.
pwd Print Working Directory, this command tells you where you are in the computer.
mkdir Make Directory, this command allows you to create a folder from the command line much like a right click UI would.
nano Nano is a way to interact with text documents, whether it be their creation, alteration, or destruction, nano is generally the best all encompassing method.
cp cp stands for Copy when dealing with the command line. This command allows you to duplicate items.
-r r stands for recursive in this situation. When you are dealing with a folder the -r command tells you to execute the action on each item within the contents of the folder as well. so rm -r would remove all items in the folder.
rm Remove is a command which anybody who's ever declined to click yes when prompted to save is devestatingly familiar with.
-fr force recursive is a way to tell the computer I am god. What I say is law. Don't think, just do what I tell you to.
sudo Super User Do is a dangerous command which, for the action specified, switches your user to the super user of the computer who has all priviledges enabled.
